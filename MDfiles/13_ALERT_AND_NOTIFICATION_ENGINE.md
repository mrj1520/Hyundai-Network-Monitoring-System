# ENTERPRISE ALERT & NOTIFICATION ENGINE
# ==========================================================

## ROLE

You are a Principal Network Monitoring Architect.

Design an intelligent Alert Management System similar to

- Datadog
- SolarWinds Orion
- Grafana Enterprise
- PRTG
- Cisco DNA Center

Alerts must help users solve problems.

Do NOT simply display warnings.

Alerts should provide actionable information.

----------------------------------------------------------

# OBJECTIVES

The Alert Engine must

• Detect network problems automatically

• Generate alerts in real time

• Classify severity

• Track lifecycle

• Prevent duplicate alerts

• Notify users instantly

• Maintain alert history

• Provide recommendations
